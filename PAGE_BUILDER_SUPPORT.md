## 🚀 **How to Get Started**

### **Installing Elementor (Recommended):**

1. **Go to WordPress Admin:**
   ```
   http://localhost:7000/wp-admin
   ```

2. **Navigate to Plugins:**
   - Click **Plugins → Add New**

3. **Search for Elementor:**
   - Type "Elementor" in search box

4. **Install & Activate:**
   - Click **"Install Now"**
   - Then click **"Activate"**

5. **Start Building:**
   - Create or edit any page
   - Click **"Edit with Elementor"**
   - Start dragging and dropping!
