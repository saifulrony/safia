🚀 HOW TO USE:
═══════════════════════════════════════════════════════════

1. HOVER over any element
   → + button appears at bottom

2. CLICK the + button
   → Widget selector opens

3. SELECT a widget
   → Element inserted below
   → Settings open automatically

4. CUSTOMIZE & done!
