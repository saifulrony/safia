🚀 EASY WAY (Just Use Address):

1. Add Map widget to your page

2. Click "Edit" button

3. In the Location tab, enter ONLY the Address:
   
   Examples:
   ✓ "Eiffel Tower, Paris"
   ✓ "1600 Amphitheatre Parkway, Mountain View, CA"
   ✓ "Big Ben, London"
   ✓ "Sydney Opera House, Australia"
   ✓ "Central Park, New York"
   ✓ "Tokyo Tower, Japan"

4. Leave Latitude and Longitude EMPTY (or delete them)

5. Done! Google Maps will find it automatically!

═══════════════════════════════════════════════════════════

⚙️ HOW IT WORKS:

OPTION 1: ADDRESS ONLY (Easy!)
├─ Type address in "Address" field
├─ Leave "Latitude" and "Longitude" empty
└─ Google Maps geocodes it automatically

OPTION 2: COORDINATES (Precise)
├─ Enter Latitude
├─ Enter Longitude
├─ Address is just for display text
└─ Map uses exact coordinates
