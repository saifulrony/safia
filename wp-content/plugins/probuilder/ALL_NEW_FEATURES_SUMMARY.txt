3 & 4. ✨ TEXT PATH FEATURE - CURVED TEXT!
═══════════════════════════════════════════════════════════

AVAILABLE ON:
  ✓ Heading Widget
  ✓ Text Widget

3 PATH TYPES:
  1. Curve (Arc) - Simple curved path
  2. Wave - Wavy flowing path
  3. Circle - Circular arc

CONTROLS:
  ✓ Enable Text Path toggle
  ✓ Path Type selector
  ✓ Curve Amount: -100 to +100
     Positive = curve UP
     Negative = curve DOWN

VISUAL EXAMPLE:
        T h i s   i s   C u r v e d
     ╱                                ╲
   ╱                                    ╲

PERFECT FOR:
  • Hero section headings
  • Banners
  • Creative headers
  • Decorative text
  • Logos and badges

HOW TO USE:
  1. Add Heading or Text widget
  2. Go to Style → Text Path & Effects
  3. Enable Text Path: ON
  4. Choose path type
  5. Adjust curve amount
  6. Beautiful curved text!
